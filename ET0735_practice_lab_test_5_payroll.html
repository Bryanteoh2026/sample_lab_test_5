Part B: Python Programming

  1. Your company has been awarded a contract to develop an "Employee Payroll System" to calculate staff salaries with overtime and seniority rules.
  2. Files provided:
    • labsample5.py — Main Python script
    • test_labsample5.py — PyTest cases
    • employees.csv — Employee records (list of dictionaries)
  3. A helper function is already fully implemented — study it carefully as your functions must use it:
    • calc_gross_pay(employee) — computes gross pay based on the rules below
    Gross pay rules:
    1. First 40 hours: hours_worked × hourly_rate
    2. Overtime (hours > 40): overtime_hours × hourly_rate × 1.5
    3. Senior bonus: if is_senior == True, multiply total by 1.10
  4. Implement REQ-01 to REQ-04 as defined in Table 1 below.
Requirement IDDescriptionPython Function
REQ-01 When the user selects Option 4 "Display employees by department", return a list of employee dictionaries whose department matches the given parameter. Match must be case-insensitive. Return [] if none found or list is empty. get_employees_by_department(employees, department)
REQ-02 When the user selects Option 3 "Display highest paid employee", return the employee dictionary with the highest computed gross pay. Return None if the list is empty.
Hint: call calc_gross_pay() to compare employees.
 get_highest_paid_employee(employees)
REQ-03 When the user selects Option 2 "Display statistics", compute the total payroll cost — the sum of gross pay across all employees. Return 0 if the list is empty. calc_total_payroll(employees)

Hint: group employees by department first, then compute the average gross pay for each group.
